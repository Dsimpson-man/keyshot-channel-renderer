# KeyShot 通道渲染工具

一键渲染 KeyShot 场景的所有通道图（Alpha、阴影、反射、法线等），支持分辨率预设、质量控制方式及常见输出格式。  
**仅需在 KeyShot 渲染面板中勾选所需通道（需选择“单独文件”模式），脚本即可自动完成渲染并清理原始图片，只保留通道文件。**

## ✨ 功能特点

- 分辨率预设：4K / 8K / 自定义宽高
- 质量控制方式：
  1. 采样数（自动切换为 CPU 渲染器）
  2. 时间限制（秒，GPU/CPU 通用）
  3. 不控制（使用 KeyShot 界面当前设置）
- 输出格式：PNG / EXR / TIFF
- 自动删除渲染生成的主图片，仅保留通道文件（文件名前缀保持不变，通道名追加在后面）
- 基于 KeyShot 内置 `lux` API 开发，无需安装额外依赖

## 📋 系统要求

- **KeyShot 9 或更高版本**（支持 `lux.renderImage` 及通道单独输出）
- 操作系统：Windows / macOS（KeyShot 支持的平台）

## 🚀 使用方法

1. 在 KeyShot 中打开你的 3D 场景。
2. 打开 **渲染 > 渲染选项**，在“通道”面板中勾选你需要输出的通道（如 Alpha、阴影、法线等）。
   - **关键设置**：输出模式必须选择 **“单独文件”**（每个通道生成独立文件）。
3. 运行本脚本：
   - 在 KeyShot 中通过 **“脚本控制台”** 或 **“脚本编辑器”** 载入 `渲染通道-keyshot.py` 并执行。
   - 或通过 `lux.executeFile()` 方式调用。
4. 在弹出的对话框中设置：
   - 分辨率
   - 质量控制方式
   - 输出格式
   - 输出文件夹及文件名前缀
5. 脚本将自动：
   - 应用质量控制设置
   - 渲染主图与所有通道
   - 删除主图文件，保留通道文件

## ⚙️ 参数说明

| 参数             | 说明                                                                 |
| ---------------- | -------------------------------------------------------------------- |
| 分辨率预设       | 4K / 8K / 自定义 – 自定义时需要手动输入宽高                          |
| 质量控制方式     | 采样数（自动切 CPU） / 时间限制（秒） / 不控制（使用界面当前设置）     |
| 采样数/时间限制  | 仅在对应方式下有效                                                   |
| 输出格式         | PNG（推荐）、EXR（高动态范围）、TIFF                                |
| 输出文件夹       | 所有生成文件存放的目录                                               |
| 文件名前缀       | 如“通道图” → 通道文件会命名为 `通道图_alpha.png` 等形式                |

## 📝 注意事项

- 脚本会**自动删除**渲染生成的主图片（即你指定的 `basename + 扩展名` 文件），**请确保输出文件夹中没有同名重要文件**。
- 通道文件的命名规则由 KeyShot 决定，通常为 `前缀_通道名.扩展名`。脚本会列出所有生成的通道文件。
- 采样数模式会自动尝试将渲染引擎切换为 CPU（部分场景下 GPU 采样控制不稳定），若切换失败会给出警告但不影响渲染。
- 如果 KeyShot 版本较旧或 API 变动，某些参数（如 `setMaxTimeRendering`）可能不可用，脚本会捕获异常并继续执行。
